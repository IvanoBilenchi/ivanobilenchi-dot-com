UString ulog_level_to_string(ulog_level level)

Returns the string representation of the specified log level.

Note

You must not call ustring_deinit on the returned string.

Parameters:

  • level – Log level.

Returns:

String representation of the log level.

ulog_level ulog_level_from_string(UString string)

Converts the specified string to a log level.

Note

The match is case-insensitive and based on the beginning of the string. For example, “info” will match ULOG_INFO, and so will “INFO”, “Information”, etc.

Parameters:

  • string – String representation of the log level.

Returns:

Log level corresponding to the specified string. If the string does not match any known log level, returns ULOG_DISABLED.

void ulog_disable(ULog *log)
