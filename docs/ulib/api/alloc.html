Memory allocation

ulib_malloc(size)

Allocates size bytes of uninitialized storage.

Parameters

  • size – [size_t] Number of bytes to allocate.

Returns

Pointer to the beginning of the allocated memory, or NULL on failure.

ulib_calloc(num, size)

Allocates memory for an array of num objects of size and initializes all bytes in the allocated storage to zero.

Parameters

  • num – [size_t] Number of objects.

  • size – [size_t] Size of each object.

Returns

Pointer to the beginning of the allocated memory, or NULL on failure.

ulib_realloc(ptr, size)

Reallocates the given memory area with a new size.

Parameters

  • ptr – [void *] Pointer to the memory area to reallocate.

  • size – [size_t] New size of the memory area in bytes.

Returns

Pointer to the beginning of the allocated memory, or NULL on failure.

ulib_free(ptr)

Deallocates the given memory area.

Parameters

  • ptr – [void *] Pointer to the memory area to deallocate.

ulib_alloc(ptr)

Allocates memory to hold the type of the pointed variable.

Parameters

  • ptr – [T*] Typed pointer to the variable.

Returns

[void *] Pointer to the allocated memory area.
