int uversion_compare(UVersion lhs, UVersion rhs)

Compares lhs and rhs.

Parameters:

  • lhs – First version.

  • rhs – Second version.

Returns:

-1 if lhs is smaller than rhs, 0 if they are equal, 1 if lhs is greater than rhs.
